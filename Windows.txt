== How to build LLVM on Windows for use with GHC ==

1. Install MinGW (should be in %PATH% and before GHC)
2. Install CMake
3. Install Python
4. Download LLVM (sources)
5. Run CMake (during configuration, choose MinGW makefiles / native compiler)
6. Run mingw32-make.exe in the directory which contains the makefiles generated by CMake using a MinGW command prompt (don't run make.exe!) - this will take some time
7. Done
